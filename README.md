This project is currently in development, and the server-side implementation was created by another developer.The front end is implemented using React
Link to the server - https://github.com/rolling-scopes-school/fun-chat-server/tree/main?tab=readme-ov-file#Receiving-a-Message-From-a-User

After logging in (using simple real-time validation based on the server's existing capabilities), you can access the chat page. There, online users are loaded, and you can select a recipient to start a conversation.
